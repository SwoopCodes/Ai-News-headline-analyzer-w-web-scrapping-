News headline analyser with machine learning and selenium web scraping.

This program works by entering a stock ticker into the input field and running the program.

The process is as follows:
- Finds and scapes news headlines with Selenium driver
- gives the trained Machine learning model the scraped news headlines
- Model puts the headlines into either a good or bad category
- News headlines and model's analysis are output onto the interface
